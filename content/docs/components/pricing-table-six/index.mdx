## Features

- **Interactive Header**: Engaging title and description with user count slider
- **Dynamic Pricing Cards**: Three-tier pricing structure with popular plan highlighting
- **Interactive Slider**: Visual representation of user count with real-time updates
- **Comprehensive Comparison Table**: Detailed feature comparison across all plans
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Theme Support**: Full light/dark mode compatibility
- **Accessibility**: Built with accessibility best practices

## Props

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| `className` | `string` | Additional CSS classes for styling | - |
| `title` | `string` | Main pricing table title | "Choose a plan that's right for you" |
| `description` | `string` | Pricing table description | "We believe Untitled should be accessible to all companies, no matter the size of your startup." |
| `plans` | `Plan[]` | Array of pricing plans (see [Plan interface](/docs/interfaces#plan-interface)) | Built-in plans |
| `features` | `Feature[]` | Array of features for comparison table | Built-in features |
| `onPlanSelect` | `(planId: string) => void` | Callback when a plan is selected | - |

## Theming

The pricing table component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.
