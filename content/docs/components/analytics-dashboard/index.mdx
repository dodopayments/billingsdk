## Usage

```tsx
import { AnalyticsDashboard } from "@/components/billingsdk/analytics-dashboard";
```

## Props

### AnalyticsDashboard

| Prop | Type | Description |
| :--- | :--- | :--- |
| `data` | `AnalyticsData` | The metrics and revenue data to display. |
| `theme` | `"minimal" | "classic"` | The visual theme applied to the components. |
| `className` | `string` | Optional CSS class for the container. |

## Components

### AnalyticsCard

Animated bar/area charts showing growth trends.
