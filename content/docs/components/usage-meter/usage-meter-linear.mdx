## Usage

```tsx
import { UsageMeter } from '@/components/billingsdk/usage-meter';
```

```tsx
// Default Usage Meter Linear
<UsageMeter
  usage={[
    { name: 'API Requests', usage: 75, limit: 100 },
    { name: 'Storage (GB)', usage: 12, limit: 100 },
]}
  title="Usage"
  description="Your current usage across tracked limits"
  variant="linear"
  size="md"
  className="mx-auto"
  progressColor="default" // default, usage
/>

// Custom Usage Meter Linear
<UsageMeter
  usage={[
    { name: 'API Requests', usage: 75, limit: 100 },
    { name: 'Storage (GB)', usage: 12, limit: 100 },
  ]}
  	title="Usage"
	description="Your current usage across tracked limits"
	variant="linear"
	size="md"
	className="mx-auto"
	progressColor="usage" // default, usage
/>
```

## Props

| Prop            | Type                   | Required | Description                                                                            |
| --------------- | ---------------------- | -------- | -------------------------------------------------------------------------------------- |
| `usage`         | `Usage[]`              | ✅       | Array of usage items `{ name?: string; usage: number; limit: number }`.                |
| `className`     | `string`               | ❌       | Additional CSS classes to apply to the root card.                                      |
| `variant`       | `"linear" \| "circle"` | ❌       | Visual style of the meter. Defaults to `"linear"`.                                     |
| `size`          | `"sm" \| "md" \| "lg"` | ❌       | Component size mapping (bar/circle/text). Defaults to `"md"`.                          |
| `title`         | `string`               | ❌       | Optional title rendered in the card header.                                            |
| `description`   | `string`               | ❌       | Optional description rendered under the title.                                         |
| `progressColor` | `"default" \| "usage"` | ❌       | Bar/ring color behavior. `"usage"` auto-colors by thresholds. Defaults to `"default"`. |

## Theming

The usage meter linear component is styled using the `shadcn/ui` library.
