## Usage

```tsx
import { UsageMeterBar } from "@/components/billingsdk/usage-meter-bar";
```
```tsx
<UsageMeterBar
  title="Data Usage"
  currentUsage={75}
  maxUsage={100}
  unit="GB"
  description="You have used 75% of your data limit."
/>
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `title` | `string` | ✅ | The title of the usage meter bar |
| `currentUsage` | `number` | ✅ | The current usage value |
| `maxUsage` | `number` | ✅ | The maximum usage limit |
| `unit` | `string` | ✅ | The unit of measurement (e.g., GB, MB) |
| `description` | `string` | ❌ | Additional description for the usage meter |

## Theming

The usage meter bar component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.

```tsx
import { UsageMeterBarDemo } from "@/components/billingsdk/usage-meter-bar-demo";
```
