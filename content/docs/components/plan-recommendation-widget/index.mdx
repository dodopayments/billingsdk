## Usage

To use the Plan Recommendation Widget in your application, import the demo component:

```tsx
import { PlanRecommendationWidgetDemo } from '@/components/plan-recommendation-widget-demo';
```

Then use it in your JSX:

```tsx
<PlanRecommendationWidgetDemo />
```

For custom implementations, refer to the demo component source code which shows how to configure the widget with your own plans and usage data.

## Props

| Prop              | Type                                    | Description                                      |
| ----------------- | --------------------------------------- | ------------------------------------------------ |
| `className`       | `string`                                | Additional CSS classes to apply to the widget    |
| `currentPlan`     | `Plan`                                  | The user's current plan (optional)               |
| `plans`           | `Plan[]`                                | Array of available plans                         |
| `limits`          | `{ apiCalls?: number; storage?: number; teamSize?: number }` | Usage limits for calculating percentages (optional, defaults to Infinity for unlimited) |
| `usageData`       | `{ apiCalls: number; storage: number; teamSize: number }` | Usage data to base recommendations on |
| `onPlanSelect`    | `(planId: string) => void`              | Function called when upgrade is completed        |
| `onComparePlans`  | `() => void`                            | Function called when "Compare Plans" is clicked  |
| `onLearnMore`     | `() => void`                            | Function called when "Learn More" is clicked     |

### Plan Object

| Property        | Type     | Description                                |
| --------------- | -------- | ------------------------------------------ |
| `id`            | `string` | Unique identifier for the plan             |
| `title`         | `string` | Display name of the plan                   |
| `description`   | `string` | Brief description of the plan              |
| `currency`      | `string` | Currency symbol (e.g., "$")                |
| `monthlyPrice`  | `string` | Monthly price of the plan                  |
| `yearlyPrice`   | `string` | Yearly price of the plan                   |
| `features`      | `Array`  | List of features included in the plan      |

### Usage Data Object

| Property    | Type     | Description                              |
| ----------- | -------- | ---------------------------------------- |
| `apiCalls`  | `number` | Number of API calls made                 |
| `storage`   | `number` | Amount of storage used (in GB)           |
| `teamSize`  | `number` | Number of team members                   |

- **Clean Design**: 
  - Minimal styling without backgrounds
  - Clear visual hierarchy
  - Consistent spacing and typography

- **Usage Stats Section**:
  - Gradient progress bars with color-coded statuses
  - Detailed metrics with contextual information
  - Critical warning indicators for high usage

- **Smart Recommendation Panel**:
  - Clear value proposition
  - Prominent pricing display
  - Gradient background for visual interest

- **Clear CTA Button**:
  - Prominent "Upgrade to Pro Plan" button
  - Full-width for emphasis
  - Standard button styling

- **Secondary CTAs**:
  - Minimal outlined "Compare Plans" button
  - Subtle "Learn More" text link
