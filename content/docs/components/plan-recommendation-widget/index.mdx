## Usage

```tsx
import { PlanRecommendationWidget } from '@/components/billingsdk/plan-recommendation-widget';
```

```tsx
<PlanRecommendationWidget
  plans={plans}
  usageData={{
    apiCalls: 15000,
    storage: 50,
    teamSize: 12
  }}
  onPlanSelect={(planId: string) => console.log('Selected plan:', planId)}
/>
```

## Props

| Prop           | Type                                                                 | Description                                                   |
| -------------- | -------------------------------------------------------------------- | ------------------------------------------------------------- |
| `className`    | `string`                                                             | Additional CSS classes to apply to the widget                 |
| `currentPlan`  | `Plan`                                                               | The user's current plan (optional)                            |
| `plans`        | `Plan[]`                                                             | Array of available plans                                      |
| `usageData`    | `{ apiCalls: number; storage: number; teamSize: number }`            | Usage data to base recommendations on                         |
| `onPlanSelect` | `(planId: string) => void`                                           | Function called when a plan is selected                       |
| `title`        | `string`                                                             | Title for the widget (default: "Plan Recommendation")         |
| `description`  | `string`                                                             | Description for the widget (default: "Based on your usage...")|

### Plan Object

| Property        | Type     | Description                                |
| --------------- | -------- | ------------------------------------------ |
| `id`            | `string` | Unique identifier for the plan             |
| `title`         | `string` | Display name of the plan                   |
| `description`   | `string` | Brief description of the plan              |
| `currency`      | `string` | Currency symbol (e.g., "$")                |
| `monthlyPrice`  | `string` | Monthly price of the plan                  |
| `yearlyPrice`   | `string` | Yearly price of the plan                   |
| `features`      | `Array`  | List of features included in the plan      |

### Usage Data Object

| Property    | Type     | Description                              |
| ----------- | -------- | ---------------------------------------- |
| `apiCalls`  | `number` | Number of API calls made                 |
| `storage`   | `number` | Amount of storage used (in GB)           |
| `teamSize`  | `number` | Number of team members                   |
