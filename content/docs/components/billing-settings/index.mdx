```tsx
import { BillingSettings } from '@/components/billingsdk/billing-settings';

<BillingSettings
	title="Billing Settings"
	description="Manage your billing preferences and settings"
	initialData={{
		email: 'user@example.com',
		name: 'John Doe',
		autoRenewal: true,
		invoiceEmails: true,
		promotionalEmails: false,
		currency: 'USD',
		taxId: 'EU123456789',
	}}
	onSave={(data) => {
		console.log('Saved billing settings:', data);
	}}
	onCancel={() => {
		console.log('Cancelled billing settings changes');
	}}
/>;
```

## Props

| Prop        | Type                  | Description                                       |
| ----------- | --------------------- | ------------------------------------------------- |
| className   | `string`              | Additional CSS classes to apply to the component  |
| title       | `string`              | Title for the billing settings section            |
| description | `string`              | Description for the billing settings section      |
| initialData | `BillingSettingsData` | Initial data to populate the form                 |
| onSave      | `(data) => void`      | Function called when the form is submitted        |
| onCancel    | `() => void`          | Function called when the cancel button is clicked |

### BillingSettingsData Object

| Property          | Type      | Description                           |
| ----------------- | --------- | ------------------------------------- |
| email             | `string`  | User's billing email address          |
| name              | `string`  | User's full name                      |
| autoRenewal       | `boolean` | Whether subscription auto-renews      |
| invoiceEmails     | `boolean` | Whether to receive invoice emails     |
| promotionalEmails | `boolean` | Whether to receive promotional emails |
| currency          | `string`  | Preferred currency for billing        |
| taxId             | `string`  | Tax identification number (optional)  |
