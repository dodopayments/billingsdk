## Props

| Prop           | Type                   | Description                                                  |
| -------------- | ---------------------- | ------------------------------------------------------------ |
| className      | `string`               | Additional CSS classes to apply to the component             |
| title          | `string`               | Title for the payment methods section                        |
| description    | `string`               | Description for the payment methods section                  |
| paymentMethods | `PaymentMethod[]`      | Array of saved payment methods                               |
| onAdd          | `() => void`           | Function called when add payment method button is clicked    |
| onRemove       | `(id: string) => void` | Function called when remove payment method button is clicked |
| onSetDefault   | `(id: string) => void` | Function called when set default button is clicked           |

### PaymentMethod Object

| Property  | Type                | Description                                    |
| --------- | ------------------- | ---------------------------------------------- |
| id        | `string`            | Unique identifier for the payment method       |
| type      | `"credit" \| "ach"` | Type of payment method                         |
| brand     | `string` (optional) | Brand of the card (e.g., "Visa", "Mastercard") |
| last4     | `string`            | Last 4 digits of the payment method            |
| expiry    | `string` (optional) | Expiration date (for cards)                    |
| isDefault | `boolean`           | Whether this is the default payment method     |

```tsx
<PaymentMethodCard
	title="Payment Methods"
	description="Your saved payment methods"
	paymentMethods={[
		{
			id: 'pm_123',
			type: 'credit',
			brand: 'American Express',
			last4: '1234',
			expiry: '05/2028',
			isDefault: true,
		},
		{
       id: 'pm_456',
       type: 'ach',
-      last4: '8765',
+      bankName: 'Chase',
+      routing: '*****123',
       isDefault: false,
     },
	]}
	onAdd={() => {
		// Redirect to payment method addition flow
		router.push('/settings/billing/add-payment-method');
	}}
	onRemove={(paymentMethodId) => {
		// Remove the payment method
		removePaymentMethod(paymentMethodId);
	}}
	onSetDefault={(paymentMethodId) => {
		// Set as default payment method
		setDefaultPaymentMethod(paymentMethodId);
	}}
/>
```
