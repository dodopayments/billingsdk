## Usage

```tsx
import { PaymentMethodManager2, type PaymentMethod2 } from "@/components/billingsdk/payment-method-manager-2";
```

```tsx
const [paymentMethods, setPaymentMethods] = useState<PaymentMethod2[]>([
  {
    id: 'pm_1',
    type: 'credit',
    brand: 'Visa',
    last4: '4242',
    expiry: '12/2027',
    isDefault: true,
    cardholderName: 'John Doe',
    createdAt: '2023-01-15',
    status: 'active',
  },
  {
    id: 'pm_2',
    type: 'ach',
    bankName: 'Chase Bank',
    last4: '8765',
    isDefault: false,
    cardholderName: 'John Doe',
    createdAt: '2023-05-10',
    status: 'active',
  },
]);
```

```tsx
<PaymentMethodManager2
  className="max-w-4xl mx-auto"
  paymentMethods={paymentMethods}
  onAdd={() => console.log("Add payment method")}
  onEdit={(id, changes) => console.log("Edit payment method", id, changes)}
  onRemove={(id) => console.log("Remove payment method", id)}
  onSetDefault={(id) => console.log("Set default payment method", id)}
  onViewDetails={(id) => console.log("View details for payment method", id)}
/>
```

## Props

| Prop             | Type                                                    | Required | Description                                               |
| ---------------- | ------------------------------------------------------- | -------- | --------------------------------------------------------- |
| `className`      | `string`                                                | ❌        | Additional CSS classes for styling                        |
| `paymentMethods` | `PaymentMethod2[]`                                      | ✅        | Array of payment methods to display                       |
| `onAdd`          | `() => void`                                            | ❌        | Callback when the add button is clicked                   |
| `onEdit`         | `(id: string, changes?: Partial<PaymentMethod2>) => void`| ❌        | Callback when the edit button is clicked                  |
| `onRemove`       | `(id: string) => void`                                  | ❌        | Callback when the remove button is clicked                |
| `onSetDefault`   | `(id: string) => void`                                  | ❌        | Callback when the set default button is clicked           |
| `onViewDetails`  | `(id: string) => void`                                  | ❌        | Callback when the view details button is clicked          |
| `isLoading`      | `boolean`                                               | ❌        | Whether the component is in a loading state               |
| `error`          | `string \| null`                                        | ❌        | Error message to display                                  |

## PaymentMethod2 Interface

```tsx
interface PaymentMethod2 {
  id: string;
  type: 'credit' | 'ach';
  last4: string;
  expiry?: string; // Required for credit cards (MM/YY format)
  isDefault: boolean;
  brand?: string; // Required for credit cards
  bankName?: string; // Required for ACH accounts
  cardholderName?: string;
  createdAt: string;
  status: 'active' | 'expired' | 'inactive';
}
```

## Theming

The Payment Method Manager 2 component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.
