## Props

| Prop           | Type                   | Description                                                  |
| -------------- | ---------------------- | ------------------------------------------------------------ |
| paymentMethods | `PaymentMethod2[]`     | Array of saved payment methods                               |
| onAdd          | `() => void`           | Function called when add payment method button is clicked    |
| onEdit         | `(id: string) => void` | Function called when edit payment method button is clicked   |
| onRemove       | `(id: string) => void` | Function called when remove payment method button is clicked |
| onSetDefault   | `(id: string) => void` | Function called when set default button is clicked           |
| onViewDetails  | `(id: string) => void` | Function called when view details button is clicked          |
| className      | `string`               | Additional CSS classes to apply to the component             |
| isLoading      | `boolean`              | Whether the component is in a loading state                  |
| error          | `string \| null`       | Error message to display if there was an error               |

### Payment Method 2 Object

| Property       | Type                                  | Description                                    |
| -------------- | ------------------------------------- | ---------------------------------------------- |
| id             | `string`                              | Unique identifier for the payment method       |
| type           | `"credit" \| "ach"`                   | Type of payment method                         |
| last4          | `string`                              | Last 4 digits of the payment method            |
| expiry         | `string` (optional)                   | Expiration date (for cards)                    |
| isDefault      | `boolean`                             | Whether this is the default payment method     |
| brand          | `string` (optional)                   | Brand of the card (e.g., "Visa", "Mastercard") |
| bankName       | `string` (optional)                   | Name of the bank (for ACH accounts)            |
| cardholderName | `string` (optional)                   | Name of the cardholder                         |
| createdAt      | `string`                              | Date when the payment method was created       |
| status         | `"active" \| "expired" \| "inactive"` | Status of the payment method                   |
