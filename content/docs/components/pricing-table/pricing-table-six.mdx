## Usage

```tsx
import { PricingTableSix } from "@/components/billingsdk/pricing-table-six";
import { plans } from "@/lib/billingsdk-config";

export default function Example() {
  return (
    <PricingTableSix
      plans={plans}
      title="Image-Enhanced Pricing Cards"
      description="Hover to flip and explore features with comprehensive image support and theme-aware gradients"
      onPlanSelect={(planId) => console.log("Selected plan:", planId)}
      size="large" /* small, medium, large */
      theme="minimal" /* minimal or classic */
      className="w-full"
      imageHeight={200} /* Custom image height in pixels */

        true

      }} */
    />
  );
}
```

## Props

| Prop               | Type                             | Description                                                                    |
| ------------------ | -------------------------------- | ------------------------------------------------------------------------------ |
| `plans`            | `Plan[]`                         | Array of pricing plans (see [Plan interface](/docs/interfaces#plan-interface)) |
| `title`            | `string`                         | Main title for the pricing section                                             |
| `description`      | `string`                         | Subtitle description                                                           |
| `onPlanSelect`     | `(planId: string) => void`       | Callback when a plan is selected                                               |
| `size`             | `"small" \| "medium" \| "large"` | Size variant of the pricing table (default: "medium")                          |
| `theme`            | `"minimal" \| "classic"`         | Theme variant (default: "minimal")                                             |
| `backgroundImages` | `{ [key: string]: string }`      | Custom background images for each plan (optional)                              |
| `imageHeight`      | `string \| number`               | Custom height for image section (supports pixels and CSS values)               |
| `showFeatureTable` | `boolean`                        | Whether to show the feature comparison table below cards (default: true)       |
| `className`        | `string`                         | Additional CSS classes                                                         |

## Features

### üîÑ Interactive Flip Animation

- **Whole Card Flip**: Entire card flips on hover to reveal features
- **Preserved Pricing**: Pricing and button remain consistent on both sides
- **All Features Visible**: Complete features list displayed without scrolling
- **3D Transform**: Smooth 700ms perspective-based flip animation
- **Progressive Disclosure**: Features hidden until user interaction
- **Visual Hints**: Subtle "Hover to see features" indicator with animated arrow
- **Staggered Animations**: Feature items animate in sequence for polished feel

### üñºÔ∏è Image Integration

- **Split Layout**: Image container in top half, pricing details in bottom half
- **Dynamic Backgrounds**: Customizable background images per plan via `backgroundImages` prop
- **Animated Elements**: Floating abstract shapes with smooth animations
- **Default Images**: Includes colorful fluid/galaxy-like swirl patterns as fallbacks
- **Customizable Height**: Adjustable image height through `imageHeight` prop

### üè∑Ô∏è Pill-Shaped Badges

- **Plan Type Badge**: Positioned in top-left (e.g., "Pro", "Starter")
- **"Most Popular" Badge**: Positioned in top-right for highlighted plans
- **Styling**: Black background with white text and subtle borders

### üì± Responsive Design

- **Mobile-First**: Adapts to different screen sizes
- **Rounded Corners**: Soft radius for modern appearance
- **Dark Borders**: Subtle shadows for visual separation
- **Optimized Spacing**: Plenty of padding and whitespace

## Theming

The pricing table component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.
