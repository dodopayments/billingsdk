## Usage

```tsx
import { PricingTableSix } from "@/components/billingsdk/pricing-table-six";
import { plans } from "@/lib/billingsdk-config";

export default function Example() {
  return (
    <PricingTableSix
      plans={plans}
      title="Image-Enhanced Pricing Cards"
      description="Hover to flip and explore features with comprehensive image support and theme-aware gradients"
      onPlanSelect={(planId) => console.log("Selected plan:", planId)}
      size="large" /* small, medium, large */
      theme="minimal" /* minimal or classic */
      className="w-full"
      imageHeight={200} /* Custom image height in pixels */
      showFeatureTable={
        true
      } /* Show feature comparison table */ /* Optional: Add custom images (PNG, JPG, JPEG, WebP, GIF, BMP, TIFF) */
      /* backgroundImages={{
        'starter': 'https://images.unsplash.com/photo-1557804506-669a67965ba0',

        'enterprise': 'data:image/png;base64,iVBORw0KGgoAAAANSUhEUgA...'
      }} */
    />
  );
}
```

## Props

| Prop               | Type                             | Description                                                                    |
| ------------------ | -------------------------------- | ------------------------------------------------------------------------------ |
| `plans`            | `Plan[]`                         | Array of pricing plans (see [Plan interface](/docs/interfaces#plan-interface)) |
| `title`            | `string`                         | Main title for the pricing section                                             |
| `description`      | `string`                         | Subtitle description                                                           |
| `onPlanSelect`     | `(planId: string) => void`       | Callback when a plan is selected                                               |
| `size`             | `"small" \| "medium" \| "large"` | Size variant of the pricing table (default: "medium")                          |
| `theme`            | `"minimal" \| "classic"`         | Theme variant (default: "minimal")                                             |
| `backgroundImages` | `{ [key: string]: string }`      | Custom background images for each plan (optional)                              |
| `imageHeight`      | `string \| number`               | Custom height for image section (supports pixels and CSS values)               |
| `showFeatureTable` | `boolean`                        | Whether to show the feature comparison table below cards (default: true)       |
| `className`        | `string`                         | Additional CSS classes                                                         |

## Features

### Interactive Flip Animation

- **Seamless Card Transition**: Smooth 700ms 3D perspective-based animation that flips the entire card on hover or click
- **Dual-Sided Consistency**: Maintains pricing information and call-to-action elements on both front and back sides
- **Complete Feature Visibility**: All plan features displayed without requiring scrolling for improved user experience
- **Progressive Disclosure**: Features remain hidden until user interaction to reduce cognitive load
- **Staggered Animations**: Sequential feature item animations create a polished, professional presentation

### Advanced Image Integration

- **Flexible Layout**: Split-card design with image container occupying the top half and pricing details in the bottom half
- **Multi-Format Background Support**: Accepts customizable background images in PNG, JPG, JPEG, WebP, GIF, BMP, and TIFF formats
- **Dynamic Image Management**: Per-plan background customization via the `backgroundImages` prop with comprehensive format detection
- **Intelligent Fallbacks**: Theme-aware gradient overlays with animated abstract shapes when no custom images are provided
- **Height Customization**: Adjustable image section height through the `imageHeight` prop supporting both pixel and CSS values

### Distinctive Badge System

- **Plan Identification**: Clearly labeled plan type badges positioned in the top-left corner for immediate recognition
- **Promotional Highlighting**: "Most Popular" badges for featured plans positioned in the top-right corner with visual emphasis
- **Professional Styling**: Consistent black background with white text and subtle borders for enhanced readability

### Responsive Design & Accessibility

- **Mobile-First Architecture**: Fully responsive design that adapts seamlessly to all device screen sizes
- **Modern Aesthetics**: Rounded corners and optimized whitespace for a contemporary appearance
- **Visual Hierarchy**: Subtle shadows and borders for clear visual separation between elements
- **Touch Device Support**: Click-based card flipping for improved interaction on touch-enabled devices
- **Keyboard Navigation**: Full keyboard accessibility support for inclusive user experiences

## Theming

The pricing table component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.
