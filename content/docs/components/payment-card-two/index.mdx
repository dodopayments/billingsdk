## Usage

```tsx
import { PaymentCardTwo, type PaymentCardTwoProps } from "@/components/billingsdk/payment-card-two";
```

```tsx
   <PaymentCardTwo
    title="Complete Payment"
    brandName="Dodo payments"
    item="Subscription"
    itemAmount={146}
    currency="$"
    infoBottom="Make payment to activate subscription"
     onPay={async({cardNumber})=>{
      console.log("Card Number:", cardNumber)
    }}
    onBack={() => console.log('Back clicked')}
    />
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `title` | `string` | ❌ | Header text on the screen |
| `brandName` | `string` | ❌ | Brand label shown on the card |
| `item` | `string` | ✅ | Purchase item name shown in the list |
| `itemAmount` | `number` | ✅ | Amount for the item |
| `currency` | `string` | ❌ | Currency symbol (default: `$`) |
| `infoBottom` | `string` | ❌ | Helper text under the card input |
| `onPay` | `() => void` | ❌ | Click handler for Pay now, returns card number |
| `onBack` | `() => void` | ❌ | Click handler for back button |
| `className` | `string` | ❌ | Extra classes for root container |

## Theming

The Payment card component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.
